# Reto 1 - Despliegue en Heroku

## :dart: Objetivos

- Identificar las características del desarrollo y la integración continua.
- Aplicar principios de DevOps para la construcción de pipelines

## ⚙ Requisitos

- WebStorm
- Node.js
- Jest
- Una cuenta de heroku
